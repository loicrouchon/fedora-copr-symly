# Fedora COPR Symly

This repository holds the RPM spec for the [Symly](https://github.com/loicrouchon/symly/) command line tool and publishes it to [Fedora COPR](https://copr.fedorainfracloud.org/coprs/loicrouchon/symly/packages/)
